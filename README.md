## Overview

This project demonstrates Object-Oriented Programming (OOP) concepts using ES6 classes in React. It implements a class hierarchy consisting of Person, Student, and Teacher entities with inheritance, method overriding, and polymorphism.

The application dynamically renders class instances in a modern user interface.

---

## Aim

To implement a class hierarchy using inheritance and demonstrate polymorphism in JavaScript ES6 and React.

---

## Objectives

- Create base Person class
- Extend Student and Teacher classes
- Implement method overriding
- Demonstrate polymorphism
- Render class objects dynamically
